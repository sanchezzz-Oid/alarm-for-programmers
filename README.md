## 4. Введение

**Будильник для программистов** — это специализированное приложение, которое учитывает особенности образа жизни разработчиков: ночную активность, дедлайны и нарушенный режим сна.

**Основные возможности:**
- Расчёт оптимального времени пробуждения по фазам сна
- Интеграция с GitHub для анализа ночных коммитов
- Тёмная тема, не слепящая глаза
- Дедлайн-режим для экстренных случаев
- Статистика сна и Telegram-уведомления

Проект разрабатывается с использованием Kanban-методологии для прозрачного отслеживания прогресса.

## 5. Примеры использования

1. **При нажатии на кнопку "Установить будильник"** открывается окно выбора времени и дополнительных параметров.
2. **При включении "GitHub-режима"** приложение запрашивает токен и анализирует ночные коммиты.
3. **При расчёте идеального времени** программа предлагает проснуться через полные циклы сна (4.5, 6, 7.5 или 9 часов).
4. **При выборе мелодии** можно установить звук механической клавиатуры или саундтрек из Doom.
5. **При активации "Дедлайн-режима"** будильник звонит каждые 10 минут, пока пользователь не встанет.
6. **При пропуске будильника** отправляется уведомление в Telegram с мотивирующим сообщением.
7. **При просмотре статистики** отображается график качества сна и количество ночных коммитов.
8. **При переключении темы** интерфейс меняется на тёмную или светлую версию.
9. **При подключении Google Calendar** приложение анализирует встречи и будит раньше перед важными событиями.
10. **При долгом нажатии на экран** включается ночной режим с красным фильтром для глаз.
